# Heart Disease Dashboard – Data Analysis Project

## Project Overview

This project presents an interactive **Heart Disease Dashboard** developed using **Power BI** to analyze clinical health data and identify patterns related to heart disease survival and mortality.

The dashboard provides insights into patient demographics, health indicators, and survival rates. By visualizing the data through interactive charts and key performance indicators, the project helps understand how different factors such as age, serum sodium levels, diabetes, and gender influence heart disease outcomes.

This project demonstrates practical skills in **data visualization, healthcare data analysis, and dashboard design**.

---

# Project Objectives

The main objectives of this project are:

* Analyze survival and death rates among heart disease patients.
* Understand how **age groups** affect survival outcomes.
* Examine the relationship between **serum sodium levels and survival**.
* Analyze the impact of **diabetes on patient survival**.
* Explore survival distribution across **different genders**.
* Create an interactive dashboard for better data exploration.

* **Microsoft Power BI** – Data visualization and dashboard creation

# Dataset Description

The dataset contains clinical records of patients with heart disease.

### Key Variables in the Dataset

* Age
* Gender
* Serum Sodium Level
* Platelets
* Diabetes Status
* High Blood Pressure
* Ejection Fraction
* Death Event
* Time

These variables help analyze health conditions and survival outcomes of patients.

# Dashboard Features

The dashboard contains several interactive visualizations and key performance indicators.

### Key Metrics

# Key Insights

From the analysis, several insights were identified:

### Higher Survival Rate

Approximately **68% of patients survived**, while a smaller portion experienced mortality.

### Age Influence

Older age groups tend to show higher mortality risk compared to younger groups.

### Diabetes Impact

Patients with diabetes may have different survival patterns compared to non-diabetic patients.

### Health Indicator Trends

Clinical indicators such as serum sodium levels can influence heart disease outcomes.
